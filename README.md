# AI Content Aggregator

An AI-powered web application that collects blog articles from multiple sources, processes them, personalizes recommendations, and sends notifications.

## Project Structure

- `backend/`: FastAPI backend service.
- `frontend/`: React + Vite frontend application.
- `task.md`: Development task tracking.
- `implementation_plan.md`: Current implementation plan.

## Getting Started

### Backend
1. Navigate to `backend/`.
2. Install dependencies: `pip install -r requirements.txt`.
3. Run the server: `python -m uvicorn app.main:app --reload`.

### Frontend
1. Navigate to `frontend/`.
2. Install dependencies: `npm install`.
3. Run the development server: `npm run dev`.
